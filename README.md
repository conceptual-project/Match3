# Три в ряд / Match 3

Классическая головоломка «Три в ряд» в формате PWA — играй прямо в браузере, устанавливай на домашний экран и наслаждайся оффлайн!

## 🎮 Об игре

Собирай три и более кристалла одного цвета в ряд или столбец, меняя местами соседние камни. Набирай очки, проходи уровни и попадай в таблицу рекордов!

- **5 видов кристаллов** с градиентной отрисовкой на Canvas
- **Каскадные комбинации** — чем длиннее цепочка совпадений, тем больше бонус
- **Система уровней** — каждый уровень требует собрать определённое количество кристаллов заданного цвета
- **Таймер** — успей выполнить цель за отведённое время
- **Таблица рекордов** (Top-10) сохраняется в localStorage
- **Двуязычный интерфейс** — русский / English
- **Звуковые эффекты** на базе Web Audio API (можно отключить)
- **Автоперемешивание** — если ходов нет, поле пересоздаётся

## 🕹 Как играть

### Мышь / Тачпад
1. Кликни на кристалл, чтобы выбрать его (подсветится рамкой)
2. Кликни на соседний кристалл — они поменяются местами
3. Если после обмена образовалась линия из 3+ одинаковых — они исчезнут и ты получишь очки
4. Если обмена нет — кристаллы вернутся на место

### Сенсорный экран (мобильные устройства)
- **Тап + тап** — выбери один кристалл, затем тапни на соседний
- **Свайп** — проведи пальцем по кристаллу в нужном направлении

### Управление
| Кнопка | Действие |
|--------|----------|
| ⏸ Пауза | Поставить игру на паузу |
| EN / RU | Переключить язык |
| 🔊 | Включить / выключить звук |

## 📱 Установка как PWA

Игра является Progressive Web App — её можно установить на устройство как обычное приложение:

### Android (Chrome)
1. Открой игру в Chrome
2. Нажми на меню (⋮) → **«Установить приложение»**
3. Подтверди установку — иконка появится на домашнем экране

### iOS (Safari)
1. Открой игру в Safari
2. Нажми кнопку «Поделиться» (↑) → **«На экран Домой»**
3. Нажми «Добавить» — иконка появится на рабочем столе

### Десктоп (Chrome / Edge)
1. Открой игру в браузере
2. Нажми на иконку установки в адресной строке (или меню → «Установить приложение»)
3. Игра откроется в отдельном окне без панели браузера

## 🚀 Деплой на GitHub Pages

1. Создай репозиторий на GitHub (например, `match3-game`)
2. Скопируй содержимое папки `pwa/match3/` в корень репозитория:

```
match3-game/
├── index.html
├── manifest.json
├── sw.js
├── icon-192.png
├── icon-512.png
└── README.md
```

3. Запушь изменения:
```bash
git init
git add .
git commit -m "Initial commit: Match 3 PWA"
git remote add origin https://github.com/USERNAME/match3-game.git
git push -u origin main
```

4. В настройках репозитория: **Settings → Pages → Source → main** (ветка `main`, папка `/root`)
5. Через пару минут игра будет доступна по адресу: `https://USERNAME.github.io/match3-game/`

> **Важно:** Service Worker корректно работает только по HTTPS — GitHub Pages предоставляет его по умолчанию.
