# 🧩 Три в ряд — Match-3 PWA

Классическая браузерная головоломка «три в ряд» — работает оффлайн, без магазинов приложений.

## ✨ Особенности игры

- **Система уровней:** проходи уровни последовательно, сложность растёт
- **Цели уровня:** собирай определённое количество элементов за ограниченное число ходов
- **Таймер:** следи за временем, чтобы успеть выполнить цель
- **Пауза:** нажми `P`, чтобы поставить игру на паузу и продолжить позже
- **Сохранение прогресса:** уровни и рекорды сохраняются локально
- **Мультиязычность:** переключение EN/RU
